MERN React Admin Dashboard

### Tech Included:

-   MongoDb
-   Express.js
-   React.js
-   Node.js

### Frontend:

-   MaterialUI
-   MaterialUI Data Grid
-   Nivo Charts
-   Redux Toolkit
-   RTK Query

### Backend:

-   Node.js - runtime
-   Express.js - backend framework
-   Mongoose - MongoDb management

### To Start:

-   client - npm run start
-   server - npm run dev
